PAYGW Cash Flow Assistance Calculator
 
== Description ==

PAYGW Cashflow Boost Calculator for Australian Small Business Employers. This tool is based on the ATO guidance and the current legislation before parliament. It is subject to change and is provided as a guide.

How to show put the calculator? 
1. Go to any post or page and edit it. 
2. add the shortcode [payg_calculator] in the page and save it.
